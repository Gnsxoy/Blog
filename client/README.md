# Gnsxoy_Client

**技术分享 & 学习备忘录~**

## Run Locally

Clone the project

```bash
  git clone git@github.com:Gnsxoy/Blog.git
```

Go to the project directory

```bash
  cd Blog/client
```

Install dependencies

```bash
  pnpm install
  # or
  yarn
```

Start the server

```bash
  pnpm run client
  # or
  yarn client
```

  
## Deployment

To deploy this project run

```bash
  pnpm run build
  # or
  yarn build
```
